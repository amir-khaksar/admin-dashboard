# 🚀 Admin Dashboard – React + Redux + Font Awesome

A modern RTL Admin Dashboard built using **React** and **Redux** for state management, with **Font Awesome** for icons.

This project demonstrates user management, notifications system, and profile section in a clean and structured UI.

---

## ✨ Features

- 👥 User Management System
    - Delete User
    - View User Info
    - View Messages
- 🔔 Notification Bell with Counter
- 🌙 Dark Mode Button UI
- 🧠 Global State Management using Redux
- 🧩 Component-Based Architecture
- 🎨 Clean & Responsive Design
- 🌍 RTL Layout Support
- ⭐ Font Awesome Icons Integration

---

## 🛠 Tech Stack

- React
- Redux
- React-Redux
- Font Awesome
- CSS (Flexbox Layout)

---

## ⚙️ Installation & Setup

1️⃣ Clone the repository:

```bash
git clone https://github.com/your-username/admin-dashboard.git
```

2️⃣ Navigate into the project directory:

```bash
cd admin-dashboard
```

3️⃣ Install dependencies:

```bash
npm install
```

4️⃣ Start the development server:

```bash
npm start
```

---

## 🔧 Font Awesome Setup

If not installed, install it using:

```bash
npm install @fortawesome/fontawesome-free
```

Then import it inside your main file (index.js or main.jsx):

```js
import "@fortawesome/fontawesome-free/css/all.min.css";
```
